## Week 4: Data Visualisation with Python

This tutorial focuses on data visualisation using **Pandas**, **Matplotlib**, and **Seaborn** libraries in Python Notebooks.

### Topics Covered

- **Comparison Charts** – Bar charts for comparing categorical data
- **Distribution Plots** – Histograms, KDE plots, and box plots
- **Relationship Plots** – Scatter plots, pair plots, and heatmaps
- **Composition Charts** – Pie charts

### Tasks Overview

| Task | Description                                                   | Dataset               |
| ---- | ------------------------------------------------------------- | --------------------- |
| 1    | Create comparison, distribution, and relationship plots       | Student & Search data |
| 2    | Compare movie ratings using bar charts                        | Custom movie data     |
| 3    | Visualise IQ scores with histograms and box plots             | `IQ.csv`            |
| 4    | Compare IQ scores across groups and handle outliers           | `IQ.csv`            |
| 5    | Display COVID-19 daily infections with bar chart              | Given data            |
| 6    | Analyse housing data with scatter plots, pairplots & heatmaps | `Property.csv`      |
| 7    | Create a composition chart (pie chart)                        | Custom data           |

### Data Files

- `Data/Property.csv` – Housing property data
